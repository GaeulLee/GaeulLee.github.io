온라인 시험 2023-01-31 ~ 2023-02-14

2차 개인 프로젝트(배포중)

개발인원 : 1명

개발내용
Spring MVC 방식으로 온라인 시험을 구현하였습니다.
관리자와 학생, 강사가 있으며,
관리자는 관리자, 학생, 강사 삭제 및 등록이 가능합니다.
강사는 시험 제출이 가능하고, 학생은 시험 조회가, 답안 등록이 가능합니다.

담당기능구현
사원 페이지 : 사원 등록, 삭제 가능하나 정보수정은 안 됨
강사 페이지 : 강사 비밀번호 변경, 시험회차, 문제, 보기 CRUD
학생 페이지 : 학생 비밀번호 변경, 시험회차, 문제, 보기 조회, 답안 등록
학생 점수 확인, 비동기 통신을 활용한 아이디 중복검사

개발환경
Language HTML5, CSS3, Java, SQL, JavaScript, Ajax
Framework SpringBoot
Library JSTL, JQuery, Lombok, MyBatis, Maven, BootStrap
Database MariaDB
WAS Apache Tomcat9

쇼핑몰 2023-01-04 ~ 2023-01-26

세미 팀 프로젝트(배포중)

개발인원 : 3명

개발내용
모델2 방식을 사용하여 쇼핑몰을 구현하였습니다.
쇼핑몰 내에서 관리자, 회원, 비회원의 기능별 차이를 두었고,
관리자에서도 세 등급으로 나누어 등급별 관리 권한을 나누었습니다.
관리자는 상품, 주문상태, 공지, 문의, 리뷰관리, 고객리스트조회가 가능합니다.
일반회원은 상품구매, 리뷰작성, 장바구니, 공지 조회, 정보변경이 가능하고,
비회원은 공지, 상품, 리뷰조회만 가능합니다.
외에 포인트 기능, 필터와 리스너를 활용한 방문자 수 카운트 등
다양한 기능을 구현하였습니다.

담당기능구현
쇼핑몰 메인 : 상품 정렬, 페이징, 검색
상품 상세페이지 : 해당 상품 상세정보 및 리뷰 조회
바로구매 주문 페이지 : 포인트 사용, 주소 조회/추가
주문 내역 페이지 : 관리자, 회원 기능 구분
리뷰CRUD(관리자는 본인이 등록한 상품에 한해서 리뷰 조회 가능),
주소CRUD(배송지 변경에서 CRUD 가능 및 주문 페이지에서도 주소 추가 가능)


개발환경

Library JSTL, JQuery, BootStrap
Database MariaDB
